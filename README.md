# YouDub-webui 免費免驗證版\n\n這是一個經過修改的 YouDub-webui 版本，旨在提供最簡單、完全免費且無需驗證的使用體驗。\n\n## 主要特點\n- **完全免費**：集成 Edge-TTS，無需字節跳動 API 密鑰即可進行語音合成。\n- **免驗證**：無需信用卡或身份驗證，只需 OpenAI API Key 即可開始翻譯。\n- **簡單配置**：內置圖形化設置界面，動態更新配置。\n\n## 安裝說明 (Windows)\n1. **使用安裝程序 (推薦)**：\n   - 下載本倉庫並解壓。\n   - 雙擊運行 `installer.py` (需先安裝 Python)。\n   - 按照提示選擇安裝路徑，程序會自動完成部署並創建桌面捷徑。\n\n2. **手動安裝**：\n   - 運行 `setup_windows.bat` 安裝依賴。\n   - 運行 `run_windows.bat` 啟動程序。\n\n## 使用指引\n1. 啟動後進入「設置 (免驗證)」標籤頁。\n2. 填入您的 OpenAI API Key。\n3. 確保 TTS 引擎選擇為 `edge-tts`。\n4. 點擊「更新配置」後即可開始使用。
